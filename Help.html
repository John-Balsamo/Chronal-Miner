Chronal Miner is an open source javascript based Monero mining program that allows you to mine using a Coinhive account with the aid of timers. In order to set up Chronal Miner, simply enter your public key into the section titled 'Wallet'. Once your public key is set, mining can be initiated by either hitting the manual start button on the main page, or by setting a timer with the timer button. Once the timer is set, the program will begin once the set time is reached. Mining can be stopped at any time by hitting the stop button.
